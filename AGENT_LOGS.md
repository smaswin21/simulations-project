# Agent Conversation Logs

This document contains transcripts from agent simulations in the **Commons Dilemma** game—a test of how agents cooperate or compete when managing shared resources.

## Game Overview

| Aspect | Detail |
|--------|--------|
| **Type** | Commons Dilemma / Tragedy of the Commons |
| **Setting** | Village Council managing shared pasture |
| **Initial Resources** | 120 units of grass |
| **Regeneration** | 12 units per round (stops if stock ≤ 20) |
| **Number of Rounds** | 16 rounds per simulation |
| **Agent Roles** | Scouts (monitor), Herders (graze), Regulators (enforce) |
